Agent Skill

Pitch

Turn any project into a brand‑styled deck and an AI‑narrated explainer video, with a live demo of the actual UI. Pick a variant from the slash menu.

Install

Run this once in your terminal. Pick your agent, or leave it on Both to auto-detect and install into whichever it finds.

curl -fsSL https://github.com/aadithva/pitch/releases/latest/download/install.sh | bash

Claude → ~/.claude/skills/pitch  ·  Copilot CLI → ~/.copilot/skills/pitch

Pick a variant

Open your agent (Claude or GitHub Copilot CLI) in any project and run one of these. Tap a row (or scroll) to preview each:

/pitchFull, live demo + deck + narrated video
/pitch-deckDeck only, branded slides (HTML + PDF)
/pitch-videoDeck + AI voiceover → narrated MP4
/pitch-scriptScript only, the spoken narration / talk-track
/pitch-demoLive UI demo, run the app, capture screens + a recording

How it works

Five layers, run by your agent:

01

Extract style

Pulls the project's real colors, fonts, and logo into an on‑brand theme.

02

Live demo

Runs the actual app and records the real UI & user flow.

03

Generate deck

Writes a self‑explanatory deck and renders it as themed reveal.js.

04

Narrate

Writes & speaks a script with AI voice (edge‑tts), free and local.

05

Render video

Captures the deck deterministically and muxes the voiceover into an MP4.
