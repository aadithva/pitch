Agent Skill

deck‑to‑video

Turn any project into a brand‑styled deck and an AI‑narrated explainer video — with a live demo of the actual UI. One skill, run by your agent.

Install

Run this in your terminal. It drops the skill into your skills directory and installs its dependencies.

curl -fsSL https://vibehub.microsoft.com/app/deck-to-video/install.sh | bash

Installs to ~/.claude/skills/deck-to-video  ·  override with SKILLS_DIR=…

What it does

Point your agent at a project and ask for a deck or video. The skill runs five layers:

01

Extract style

Pulls the project's real colors, fonts, and logo into an on‑brand theme.

02

Generate deck

Writes a self‑explanatory deck and renders it as themed reveal.js.

03

Narrate

Writes & speaks a script with AI voice (edge‑tts), free and local.

04

Render video

Captures the deck deterministically and muxes the voiceover into an MP4.

05

Live demo

Runs the actual app and records the real UI & user flow to showcase.

Requirements

  • Node 20+ — deck + video render
  • ffmpeg — audio/video mux
  • Chromium — installed by the script
  • Python 3 — AI voiceover (edge‑tts)
